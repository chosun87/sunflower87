## 1. 요구사항 요약
- 사용자가 구글 로그인을 거쳐 메인 대시보드(`fe/src/pages/Dashboard.jsx`)에 진입했을 때, 최상단 자산 요약 섹션과 하단 보유 종목 테이블 사이에 [오늘의 AI 추천 종목] 섹션을 배치한다.
- 철저히 객관적인 정량 지표 기반의 추천임을 사용자가 인지할 수 있도록 UI를 구성한다.

## 2. 프런트엔드(`fe`) 상세 구현 지침

### A. 레이아웃 및 반응형 그리드 설정
- 위치: `fe/src/pages/Dashboard.jsx` 중앙부
- PrimeFlex의 `grid` 시스템을 활용하여 유망 종목 카드가 가로로 배열되도록 한다.
- 화면 크기에 따라 반응형으로 처리한다. (`col-12 md:col-4` 적용하여 모바일에서는 세로 배치)

### B. UI 컴포넌트 아키텍처 준수 (★중요★)
- SUN님이 수립하신 개발 표준에 따라, `primereact` 패키지에서 직접 컴포넌트를 임포트하지 않는다.
- 반드시 `@components/PrimeReact` 중앙 집중 파일에서 `Card`, `Badge`, `Button`을 구조분해할당으로 가져와 사용할 것.

### C. 썬플라워(Sunflower) 테마 포인트 적용
- 추천 카드 상단 엣지 라인에 `border-top-3 border-amber-500` 클래스를 부여하여 프로젝트 정체성(노란색 해바라기 톤)을 강조할 것.
- 매력도 점수 및 태그를 `Badge` 컴포넌트로 시각화하여 정보의 가독성을 높일 것.
