## 📌 [COMMON] 공통 요구사항 (데이터 누수 패치)
- R1 버전의 검색 API에서 **TIGER 200, KODEX 200액티브 등 ETF 종목** 및 **한화오션** 등 사명 변경 종목이 검색되지 않던 전형적인 데이터 누수 버그를 전면 패치한다.

## 🌍 [BE_TASK] 백엔드 상세 구현 지침
- **조회 기준일 동적 바인딩:** 사명 변경 동기화를 위해 `pykrx` API 호출 시 날짜를 항상 시스템 현재 날짜(`datetime.today()`)로 동적 처리하라.
- **ETF 데이터 마스터 병합 (핵심 패치):**
  - 일반 주식 마스터 사전 외에 `stock.get_etf_ticker_and_name()`를 추가 호출하여 하나의 파이썬 딕셔너리로 통합(`update()`)하라.
- **문자열 매칭 개선:** 영문 ETF 검색 시 대소문자 미매칭으로 인한 누락을 방지하기 위해 입력 검색어와 마스터 데이터를 모두 `.lower()` 처리한 후 부분 일치 알고리즘을 수행하라.
