## 🌍 [FE_TASK] 프런트엔드 상세 구현 지침

### 1. 작업 디렉토리: `fe/`

### 2. 신규 매매 등록 Form 내 '거래일시' 입력 필드 추가
- **컴포넌트 도입:** `@components/PrimeReact`에서 **`Calendar`** 컴포넌트를 구조분해할당하여 사용하라.
- **속성 설정 사양:**
  - `showTime`, `hourFormat="24"` 속성을 활성화하여 날짜뿐만 아니라 시/분/초까지 정밀 선택 가능하게 하라.
  - 사용자가 창을 열었을 때 기본적으로 현재 날짜와 시각이 인풋에 채워져 있도록 초기 State를 `new Date()`로 설정하라.
  - 전송 데이터 포맷팅: 백엔드가 정상 파싱할 수 있도록 ISO 표준 문자열 또는 `YYYY-MM-DD HH:mm:ss` 형식의 포맷터 함수를 거쳐 전달하라.

### 3. 종목코드 검색 버튼 'Loading Spin' 구현
- **상태 변수 신설:** 검색 중임을 감지하는 `isSearching` 부울(`boolean`) State를 다이얼로그 내에 선언하라.
- **인터랙션 흐름 제어:**
  1. 사용자가 종목 검색 버튼을 클릭하는 순간 `setIsSearching(true)`를 트리거하라.
  2. PrimeReact `Button` 컴포넌트의 **`loading={isSearching}`** 속성을 바인딩하라. (검색 중일 때 자동으로 버튼 내부 아이콘이 회전하는 스핀 애니메이션으로 변신함)
  3. API 통신이 완료(`then`/`catch` 또는 `finally` 블록)되면 즉시 `setIsSearching(false)`로 되돌려 버튼을 정상화하라.
